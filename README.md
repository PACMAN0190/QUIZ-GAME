# 🧠 Python Quiz Game

A simple command-line quiz game built with Python.  
The program asks multiple-choice questions, checks the user's answers, and displays the final score.

## 🚀 Features
- Multiple-choice questions (A, B, C, D)
- Instant feedback (correct / incorrect)
- Displays correct answers
- Tracks user guesses
- Shows final score at the end

## 🛠️ Technologies Used
- Python 3
